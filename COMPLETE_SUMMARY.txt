FIREBASE SETUP (required)
=========================

1. Create project: firebase.google.com
2. Enable Firestore Database (test mode for development)
3. Get config from: Settings > Project Settings > Web App
4. Copy all values and update HTML files
5. Set Firestore Rules (see FIREBASE_CONFIG_TEMPLATE.js)
6. Collection "wishes" auto-creates on first message

Storage expected:
  - Average message size: 300-500 KB (photo included)
  - 100 messages = ~40-50 MB
  - Free tier: 1 GB (plenty!)
